 [ About ]
 
Thank you for downloading this project.

This project is a modded client created specifically for Call of Duty: IW8+ (all CoD versions from MW19 onwards, including MW2019).
It is packed with a variety of features. For details, please check the top message in the console log when the project is launched.

This project is primarily being developed to test the coding of the BodHQ client.
As a result, some features are still in development, and the project is very unstable.
While use in general gameplay is permitted, we cannot respond to all feedback.
Your opinions and feedback are welcome, so please feel free to let us know.
Also, please understand that this project's primary purpose is to develop a custom GSC, so this is a modded client primarily for development purposes.

This project allows you to perform some mod operations using software called "PHA_CoD_MC_RTM_Tool," which is available in this GitHub repository.
If you plan to play, we recommend downloading the RTM tool.

We cannot respond to technical inquiries, such as those regarding the creation of custom GSCs.
Please investigate and resolve any development issues yourself.
If you would like to use the GSC Mod menu introduced on my YouTube channel, please join the Demonware Emulator Modded Clients "BodHQ" (described below) and wait for it to be released to the public.

 [ How to use ]
 
1. Download the mod client DLL from the release page above.

2. Unzip the downloaded zip file.

3. Legally obtain a supported CoD build and place the file in your game directory.

4. To change your username, open "!profile.ini" in a text editor and change the parameter next to "PlayerName=".

5. Launch the game and enjoy!
